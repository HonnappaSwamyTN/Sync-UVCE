# Sync-UVCE

A unified platform for college students — marketplace, lost & found, and complaints/queries — in one place.

## Tech Stack
- Frontend: React
- Backend: FastAPI (Python)
- Database: PostgreSQL

## Project Structure
See ARCHITECTURE.md for full details on schema, API contracts, and design decisions.

## Branching Workflow
- `main` — stable, only updated via reviewed PRs from `dev`
- `dev` — integration branch, all feature branches merge here first
- Feature branches: `module-feature-name` (e.g. `marketplace-item-crud`)

## Getting Started
1. Clone the repo: `git clone https://github.com/HonnappaSwamyTN/Sync-UVCE.git`
